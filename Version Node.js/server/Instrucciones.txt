*********************** NodeJS *************************
1.- Verificar si esta corriendo NodeJS (Respuesta: v8.9.4)
	node -v
2.- Verificar si esta corriendo npm (Respuesta: v5.6.0)
	npm -v
3.- Crear un nuevo proyecto de NodeJS
	mkdir server
	npm init (Dentro de la carpeta contenedora)
4.- Configurar datos del nuevo proyecto de NodeJS (Creará el archivo package.json)
	name: (ejercicio2) agenda
	version: (1.0.0)
	description: Evaluacion del curso - Interactuando con Bases de Datos - Version NodeJS
	entry point: (index.js)
	git repository: https://github.com/sersystems/Interactuando_con_Bases_de_Datos.git
	keywords: NodeJS MongoDB http
	author: Sergio Regalado Alessi
	license(ISC): MIT
5.- Instalar Dependencias
	npm i -S express
	npm i -S body-parser
	npm i -S mongoose (Mongoose es una herramienta de modelado de objetos MongoDB diseñada para trabajar en un entorno asíncrono.)
	npm i -S -g watchify (Actualice cualquier archivo fuente y su paquete browserify se compilará en el acto.)
	npm i -D nodemon (nodemon observará los archivos en el directorio en el que se inició nodemon, y si los archivos cambian, nodemon reiniciará automáticamente su aplicación de nodo.)
6.- Correr Servidor
	node index.js (Para cerrar el servidor oprimimos la tecla "Ctrl+C")
7.- Ejuctar base de datos
	"C:\Program Files\MongoDB\Server\3.6\bin\mongod.exe" --dbpath C:\Proyectos_NodeJS\version_node\server\data"
